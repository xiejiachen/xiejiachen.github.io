$R$ 循环

12月 11, 2015 发布在 R

在R中，也可以使用循环语句进行结构控制。
R的循环结构与其它的语法$（C++, python）$非常相似。

目录

  1. If 循环
    1. if
    2. if.else
    3. if. else if. else
  2. For 循环
  3. While 循环
  4. Repeat 循环
  5. Next, Return

If 循环

if

if.else

if. else if. else

注意:else必须跟前面的｝在同一行。因为R语言执行程序时，每行单独执行，所以else如果在下一行，R会认为语句结束，导致报错。

For 循环

for循环表示，i从1到10，依次执行。

While 循环

while循环表示在给定的条件下无限次执行，直到不符合条件停止。

Repeat 循环

使用repeat创建一个无限的循环。只有执行到break循环才会停止

Next, Return

next表示跳过。

return表示停止循环，并返回一个值。与C++类似。
